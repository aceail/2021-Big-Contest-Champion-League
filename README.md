## About Competition
- 2021 빅콘테스트 데이터분석분야 챔피언리그 스포츠테크 부문  
- 「프로야구 배럴(Barrel)」을 통한 타자 성적 예측  
- KBO에 적합한 좋은 타구(Barrel)을 새롭게 정의  
- 정의한 Barrel을 바탕으로 타자의 OPS 예측 모형 개발
